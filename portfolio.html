Cache Simulator
  • This program is the implementation a minimilized command line cache simulator in Java. The cache can perform the following: cache-read, cache-write, cache-flush, memory-view, cache-view, memory-dump and cache-dump. The supported replacement policies are: Random replacement, LFU replacement, and LRU replacement.

Airline Manager
  • This program implements Dijkstra's algorithm to determine the lowest price for getting from one airport to another. This is a personal project that was done using Java during the summer of 2020 as a means of putting my theoretical knowledge from data structures and algorithms into practice.

HP Wand
  • HPWand is a plugin for Minecraft that I developed in Java using Bukkit interfaces. This plugin features a custom item that has 40+ unique "spell" functions along with various other features, and it has accumulated over 500 downloads on spigotmc.org.

Store Manager
  • This program was developed in C++ as a part of my data structures and algorithms course. It takes two text documents, one that describes a store and one that describes customer orders, and creates a results file that prints the contents of the store before and after filling the orders as well as the status of all the orders before and after being filled.

Snake
  • This is a replica of the classic "Snake" game that I developed using Pygame inside of Python. It uses basic collision checks against position points and the window edges. The scores list is loaded in before the window opens and will save before the program closes in order to track the high scores through multiple uses.
